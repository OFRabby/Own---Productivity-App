# Own---Productivity-App
A simple yet powerful productivity app for task management and shop calculations.

App Features Summary
✅ Task Management
- Create, edit, delete tasks
- Categories (University, Wife, Personal, Tour, Promises)
- Task images with camera/gallery
- Search tasks
✅ Notifications
- Task reminder notifications at scheduled time
- Task completion notifications
- Works when app is in background
- Lock screen notifications (tap to open)
- Persistent foreground service
✅ Shop Calculator
- Purchase list mode (price × quantity)
- Max quantity calculator
✅ UI/UX
- Material Design 3
- Dark/Light theme (auto)
- Permission handling dialog
